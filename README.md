# Tweeter API

## Installation

1.  Clone the project
    `git clone https://github.com/HugoLiconV/Twiter-IA.git`
2.  Move to folder
    `cd Twitter-IA`
3.  Install dependencies
    `pip install -r requirements.txt`
4.  Create **.env** file (use .env.example as guide)

## Execute
    `python main.py`
